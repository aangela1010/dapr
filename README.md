# dapr

Tiny Node/Express service with a `/health` endpoint, packaged with Docker.

## Why this repo exists
I’m a Scrum Master and Agile Delivery Lead. I don’t write production code in my day job, but I work closely with engineers and need to stay fluent in delivery constraints, release hygiene, and reliability basics. This repo is a small proof project that helps me practice those fundamentals end to end and communicate more clearly in technical environments.

## What it includes
- Node/Express service with a `/health` endpoint
- Docker support for repeatable local setup

## Run locally (Node)
```bash
```
npm install
npm start
curl http://localhost:3000/health
